# API Call Comparison: Sequential vs Parallel

A performance comparison tool that demonstrates the difference between sequential and parallel API calls with error handling.

## Features

- ✅ **Sequential Processing**: Processes requests one by one
- ✅ **Parallel Processing**: Processes all requests simultaneously
- ✅ **Concurrency Limiting**: Bonus implementation with batch processing
- ✅ **Error Handling**: Catches and logs failures without stopping execution
- ✅ **Performance Metrics**: Measures and compares execution times
- ✅ **TypeScript**: Fully typed for safety
