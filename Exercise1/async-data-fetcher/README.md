# Exercise 1: Async Data Fetcher

A robust, type-safe async data fetching utility that handles HTTP requests with retry logic and comprehensive error handling.

## Features

- ✅ **Type-safe** with full TypeScript support
- ✅ **Retry logic** with exponential backoff (1s, 2s, 4s)
- ✅ **Error handling** for network, HTTP, and timeout errors
- ✅ **Timeout support** with AbortController
- ✅ **Custom headers** support
- ✅ **Generic responses** for type safety

## Installation

**bash**
cd async-data-fetcher
npm install
